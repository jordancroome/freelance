How to add in Gmail
  1. Open this file in Chrome (or any browser).
  2. In the Preview box below, click and drag to select the entire signature (from the orange line down to the last link).
  3. Copy (⌘C / Ctrl+C).
  4. Gmail → Settings (gear) → See all settings → General → Signature → create/edit → paste into the editor.
  5. Scroll down and click Save Changes.

If formatting strips on paste, try pasting into a new Gmail compose window first, then copy again from there into the signature field.

Logo: The signature uses a PNG sized for email (https://jordancroome.com.au/images/logo-email.png) so Gmail and other clients display it reliably. Regenerate with npm run export:logo-email after changing images/logo.svg.
